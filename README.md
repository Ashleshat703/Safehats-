### Features

- Built with React and Vite
- Role-based authentication flow
- Separate routes for company and user login
- Home page and authentication pages
- Page not found fallback route
- Modular frontend structure

## Tech Stack

- React
- Vite
- React Router
- React Redux
- JavaScript
- CSS

## Main Routes

- `/` - Home page
- `/auth` - Authentication page
- `/auth/company` - Company authentication
- `/auth/user` - User authentication
- `*` - Page not found

## Project Structure

```text
safehats-main/
|-- src/
|   |-- App.jsx
|   |-- App.css
|   |-- assets/

|-- package.json
|-- vite.config.js

```` 
### Getting Started
Clone or download this repository
Install dependencies using:
npm install
Start the development server:
npm run dev
